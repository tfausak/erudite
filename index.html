Installation

Add it to your Gemfile:

gem 'erudite', '~> 0.3.0'

Or install it manually:

$ gem install erudite --version '~> 0.3.0'

Usage

# example.rb

# >> x = 1
# => 1
# >> x + 1
# => 2

# >> x = 2
# => 2
#
# x + 1
# => 3

# >> x
# NameError: ...

# >> f(3)
# => 9
def f(x)
  x**2
end
$ erudite example.rb
